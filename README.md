##  Descripción

Este laboratorio tiene como objetivo practicar la lectura y escritura de archivos en Python utilizando:

* Context managers ("with open")
* Pandas DataFrames
* Archivos Pickle

También se analizan conceptos básicos dentro de un pipeline de datos.

## ¿Cómo ejecutar el proyecto?

### Opción 1: Google Colab 

1. Abrir el archivo ".ipynb" en Google Colab
2. Ir a:
   Entorno de ejecución → Ejecutar todo
3. Ver los resultados directamente en el notebook

### Opción 2: Local 

1. Instalar dependencias:
   pip install pandas
2. Abrir Jupyter Notebook:
   jupyter notebook
3. Abrir el archivo ".ipynb"
4. Ejecutar todas las celdas

##  Contenido del laboratorio

###  Ejercicio 1: Archivos TXT y CSV

* Lectura y escritura de archivos usando "open()" con context managers
* Permite manejar archivos de forma segura y automática

### Ejercicio 2: Pandas DataFrames

* Lectura de archivos CSV con "pandas"
* Escritura de DataFrames a CSV
* Uso de estructuras tabulares para análisis de datos

### Ejercicio 3: Pickle

* Guardar objetos de Python en archivos ".pkl"
* Cargar objetos previamente guardados
* Útil para persistencia de datos y modelos

## Conceptos clave

* **Lectura (Read)**: usada al inicio del pipeline de datos
* **Escritura (Write)**: usada al final para guardar resultados
* **DataFrames**: ideales para datos estructurados
* **Pickle**: permite guardar objetos completos de Python
